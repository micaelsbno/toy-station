## Description

[Nest](https://github.com/nestjs/nest) starter-based exercise on railway interlocking.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```

## Developing

For features, write / read the rfcs on docs. Then write tests based on them and implement code.
